# 批量条形码生成系统

## 简介
这是一个基于Web的批量条形码生成系统，可以快速生成和打印多个条形码。

## 使用方法

### 方法一：直接打开启动脚本（推荐）

#### macOS用户：
1. 双击 `启动应用.command` 文件
2. 系统会自动启动一个HTTP服务器
3. 在浏览器中访问 `http://localhost:8000/` 即可使用

#### Windows用户：
1. 双击 `启动应用.bat` 文件
2. 系统会自动启动一个HTTP服务器
3. 在浏览器中访问 `http://localhost:8000/` 即可使用

### 方法二：手动启动

1. 打开终端或命令提示符
2. 进入项目所在目录
3. 运行命令：`python -m http.server 8000`（Windows）或 `python3 -m http.server 8000`（macOS/Linux）
4. 在浏览器中访问 `http://localhost:8000/` 即可使用

## 功能特性

- 支持批量生成条形码
- 支持多种条形码类型
- 实时预览生成的条形码
- 支持播放和显示条形码
- 可以自定义条形码宽度和高度

## 注意事项

- 需要安装Python 3.x
- 确保8000端口没有被其他程序占用
- 如果端口被占用，可以修改启动脚本中的端口号
